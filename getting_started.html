Packages

Language Requirement(s): Python. Usage Requirements: Python-compatable IDE, Jupyter Notebooks

This fairly plug-and-play Python package allows the user to run popular hadronic collision event generators/simulations. What are possible outcomes of proton-proton collisions at a certain energy? Run PYTHIA 8.3, SIBYLL-2.3d, EPOS-LHC, QGSJet-III, and more without the cumbersome FORTRAN. Chromo's authors provide a variety of examples to play around with.
