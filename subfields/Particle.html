Language Requirement(s): Python. Usage Requirements: Python-compatable IDE, Jupyter Notebooks

One of my personal favorites. This fairly plug-and-play Python package allows the user to run popular hadronic collision event generators/simulations. What are possible outcomes of proton-proton collisions at a certain energy? Run PYTHIA 8.3, SIBYLL-2.3d, EPOS-LHC, QGSJet-III, and more without the cumbersome FORTRAN. Chromo's authors provide a variety of examples to play around with.

Contibutor: Amelia Binau, Indiana University - Bloomington

Datasets

Language Requirement(s): Anything compatable with HDF5 file formats. Usage Requirements: Internet connection, compiler compatable with relevant language.

Designed specifically for broad participation in MicroBooNE analysis, this dataset contains one million muon neutrino interactions and one hundred thousand electron neutrino events in the MicroBooNE liquid argon neutrino detector. As stated in this paper, these simulated data are the same as those used in many MicroBooNE physics analyses. These data are saved as HDF5 files for both the muon and electron neutrino samples.

Contributor: Amelia Binau, Indiana University - Bloomington

Usage Requirements: Internet connection for online database.

Used by physicists around the world, the Particle Data Group (PDG) maintains and publishes summary tables of particle masses, charges, full decay widths, decay modes, and more for fundamental and composite particles. Additionally the PDG contains tables for physical constants and reviews for the Standard Model, detectors, probability/ statistics, and major experimental results.

Contibutor: Robert Laughlin, Indiana University - Bloomington
