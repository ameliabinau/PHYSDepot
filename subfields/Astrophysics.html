Usage Requirements: Fairly strong internet connection, decent graphics processing for downloadable software.

Using either the web-based interactive or the downloadable application, explore the night sky using this well-known software detailing the celestial sphere. Great for outreach, education, amateur astronomy, and anyone who wants to take a zoomed-in look at the night sky. Use this tool to determine exactly what the sky will look like at a specific location and time and to gain an interactive understanding of how to use popular astronomical coordinate systems. Stellarium allows you to chart magnitude, position, and other physical properties of almost all astronomical objects.

Contibutor: Amelia Binau, Indiana University

Usage Requirements: Strong internet connection.

Universe Monitor is a free public website created by Dave Owen, AKA "Space Dave" of Te Awamutu, New Zealand. Owen is also the creator of the Te Awamutu Space Centre, a space museum and education center. The website contains live feeds, video, CGI visualizations, text reports, etc. from various sources, such as the Solar Dynamics Observatory, NASA Earth Observatories, NOAA, ISS, etc, and includes recent space news, launches, and random astronomical facts. It is accessible from any device. Dave Owen writes that anyone is able to access the website for educational purposes, as well as set up public displays for science museums, observatories, and schools. To help keep the website ad-free, one can support their Patreon and learn more about the service on their about page.

Contibutor: Robert J. Mailliard, Indiana University - Bloomington

Usage Requirements: Works best on Linux, but also available on MacOS and Windows

How do astronomers process, analyze, and display the electromagnetic data they collect through telescopes from astronomical objects? One of the most common software for this purpose is SAODS9, a product of the Chandra X-Ray Center (CXC). This software requires astronical data to be contained in FITS (Flexible Image Transport System) files, which is the standard among astronomers. NASA has collected public FITS files for processing here and more are available on MAST. More informagion on other image viewers is provided here.

Contibutor: Amelia Binau, Indiana University

Language Requirement(s): Terminal, Bash, Fortran 90. Usage Requirements: Ubuntu Linux setup for windows, or IOS alternative, git packages

MESA is a hydrodynamic code that simulates stars, black bodies, and supernovae remnants. It is often used to simulate stars along the main sequence. The applications are wide in astrophysics and nuclear physics. You can use it to simulate the life and conditions of just about any star or black body you want. MESA also hosts a yearly summer school; past summer schools have their materials published online.

Contibutor: Aydian Brown, Purdue University

Language Requirement(s): Terminal C-Based Language. Usage Requirements: Ubuntu Linux setup for windows or IOS alternative

DISCO is a moving-mesh hydrodynamics code that simulates gases moving and evolving at its core. This can be applied to any problem in astrophysics and broader applications for other hydrodynamic/fluid dynamic problems. For example, one could simulate a pulsar or a fluid-based experiment in a lab. The basics are set up for you in templates and you can create any fluid dynamic simulation you want simply by utilizing a little basic C and reading the source code.

Contibutor: Aydian Brown, Purdue University

Language Requirement(s): Python. Usage Requirements: Python-Compatable IDE, Jupyter Notebook

galpy allows the user to model galactic dynamics or any large-scale system in a gravitaitonal potential. With plenty of examples and documentation, users can plot rotation curves and integrate orbits (plot object paths over time).

Contibutor: Amelia Binau, Indiana University - Bloomington
